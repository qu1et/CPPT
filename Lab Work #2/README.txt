Вариант 1

В данной папке содержатся четыре программы, входящие в состав первой лабораторной работы.

- Первая программа решает пример. Значения вводятся в консоль, также предусмотрен проверка деления на ноль.

- Вторая программа выводит все четные элементы списка (заданного разработчиком) построчно.

- Третья программа вычисляет сумму всех чисел из списка (заданного разработчиком), которые больше десяти.

- Четвертая программа выводит максимальное число из списка (заданного разработчиком).

- Имеется документация разработчика, поясняющая работу программ.

- Имеется документация пользователя, поясняющая работу с программой.
